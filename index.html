Services

News App

I developed a news app that fetches real-time news articles from an external API and displays them in an organized, user-friendly interface. I used HTML, CSS, and JavaScript to build the front end, and the API provided the latest news articles, which I dynamically displayed based on categories like Health, Technology, and Sports. The app is fully responsive and works well on both desktop and mobile devices. This project improved my skills in API integration and asynchronous JavaScript. In the future, I plan to add a personalized user login feature and implement pagination to handle larger datasets.

TIC-TAC-TOE Game

I developed an interactive Tic-Tac-Toe game that allows two players to take turns placing X and O marks on a 3x3 grid. The game is built using HTML, CSS, and JavaScript. The HTML structures the grid, CSS styles it, and JavaScript handles the core game logic, including switching turns, checking for winning combinations, and detecting draws. The game features a reset button, allowing players to start a new game at any time. One challenge was ensuring the win and draw detection worked correctly after each move, which I managed by implementing condition checks within the game logic. This project helped me improve my JavaScript skills, especially in handling user events and building interactive features.
